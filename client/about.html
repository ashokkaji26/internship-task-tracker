How to Use This Application

A simple step-by-step flow designed for students and interns.

  1. Register as a user by entering your name and email. Once registered, your session becomes active.
  2. Add internship-related tasks such as assignments, learning goals, or deadlines.
  3. View and manage tasks in your dashboard, including updating or deleting tasks.
  4. Track progress and stay organized throughout your internship journey.

About the Project

A full-stack project built to demonstrate real-world development skills.

The Internship Task Tracker is a full-stack web application designed to showcase practical frontend and backend integration. It focuses on clean architecture, RESTful APIs, database relationships, and a responsive, user-friendly interface.

  • REST API design with proper user–task relationships
  • MongoDB data persistence using Mongoose models
  • Dynamic frontend updates using JavaScript
  • Clean UI built with modern HTML & CSS

Tech Stack: HTML, CSS, JavaScript, Node.js, Express.js, MongoDB
